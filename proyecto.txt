Videocursos API

El alumno deberá crear una API para una web de cursos

Es importante que valides el correcto funcionamiento de tu aplicación. Debes de cumplir con los 
requerimientos indicados, las entregas y ten en cuenta las consideraciones listadas. Así mismo 
aprovecha los recursos recomendados.

Instrucciones

Genera las relaciones necesarias entre las tablas

Recuerda crearlas en el archivo initModels dentro de la carpeta models
Cuando los modelos estén creados y tengas tu conexión a la base de datos genera las rutas necesarias 
para obtener la siguiente información

Obtener a un usuario por su id (Deberás mostrar la información del usuario menos created_at, updated_at y password)
Obtener a un usuario junto con los cursos donde esta suscrito (Mostrar los mismos datos del usuario que 
en el punto anterior junto al nombre del curso)
Crear un usuario
Actualizar un usuario, (solo se permitirá los nombres y la contraseña)
Agregar un curso a un usuario
Obtener todos los cursos
Obtener todos los cursos junto a sus categorías y sus videos (de las categorías solo deberás mostrar el nombre, 
de los videos solo deberás mostrar el nombre y su url
Crear un nuevo curso
Editar un curso (solo se permite cambiar la descripción)
Crear videos
Eliminar videos
Crear categorías
Eliminar categorías
Crea los controladores necesarios para las peticiones en las rutas del punto 2.

Crea los Servicios necesarios para responder a las peticiones del punto 2.
